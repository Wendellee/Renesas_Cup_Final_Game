# 资源配置 模块引脚配置
RA8P1_HMI Board的Bsp包自带一些模块引脚配置,例:常规SDAM,以及不同于EK板的SDHI配置,反而EK板有一些多余的引脚模块例如PDM(脉冲密度调制),ESWM_RGMII(千兆以太网),SSI(串行外接口,主要用于音频)
 
**BSP默认配置模块:**
 1. [SDAM](#sdam) (常规SDAM )
 2. [SDHI](#sdhi) (4wireSD卡接口)
 3. [QSPI](#qspi) (四线SPI)
 4. [USB_FS](#usb_fs) (USB全速接口)
 5. [USB_HS](#usb_hs) (USB高速接口)
 6. [SWD](#swd) (SWD调试)
 7. 
**目前根据功能需要配置的模块:** 
- [glcdc-for-display-800-480](#glcdc-for-display-800-480)(800*480显示屏 RGB888 )
- [i2c1-for-camera-ov5640](#i2c1-for-camera-ov5640)(摄像头寄存器控制)
- [i2c0-for-screen-touch](#i2c0-for-screen-touch)(触摸屏寄存器控制)
- [sdhi-for-sd-card](#sdhi-for-sd-card)(外接SD卡存储)
- [spi-for-nrf24l01-as-real-time-video-transmitter](#spi-for-nrf24l01-as-real-time-video-transmitter)(无线传输摄像头数据 无线图传)
- [uart-for-debug](#uart-for-debug)(调试串口 输出日志)
- [uart-for-da16200-wifi](#uart-for-da16200-wifi)(DA16200 WIFI模块控制)
- [other-uart](#other-uart)(其他串口 串口6 串口9)
- [gpt-pwm](#gpt-pwm)(PWM输出 一共5组)

# 默认资源配置 模块引脚配置
# sdam
RA8P1支持的SDAM的引脚是固定的,且HMI_Board与EK板的引脚相同 无参靠价值 故不列写
# sdhi
此为HMI_Board板的默认引脚配置 为板上自带的4wireSD卡接口引脚 参见功能资源配置 [SDHI for SD Card]( #sdhi-for-sd-card )
# qspi
此为HMI_Board板的默认引脚配置 为板上自带的四线SPI引脚 不可修改 QSPI_flash 为32MB大小
|引脚|功能|备注|
|---|---|---|
|P808|QSPI_SCK|QSPI时钟引脚|
|P107|QSPI_CS|QSPI片选引脚|
|P100|QSPI_D0|QSPI数据引脚0|
|P803|QSPI_D1|QSPI数据引脚1|
|P103|QSPI_D2|QSPI数据引脚2|
|P101|QSPI_D3|QSPI数据引脚3|
